# 🎯 Como Obter Token Railway em 3 Passos

## PASSO 1: Abrir Railway Dashboard

Cole esta URL no navegador:
```
https://dashboard.railway.app/account/tokens
```

(Se aparecer erro "Not Found", faça login primeiro em https://railway.app)

---

## PASSO 2: Criar Token

1. Clique no botão **"Create Token"** (canto superior direito)
2. Dê um nome: `octa-deploy-hermes`
3. Clique **"Create"**

---
